# Assignment11-DATA-607

## Overview

This repository contains the complete work for **Assignment 11 – Personalized Recommender System** for **DATA 607**.

The objective of this assignment is to:

* Build a **personalized recommendation system** using survey-based movie rating data
* Move beyond the previous **Global Baseline Estimate** (non-personalized)
* Implement a **collaborative filtering algorithm**
* Generate **user-specific Top-N recommendations**
* Evaluate model performance using appropriate metrics

* `movies` — movie titles and release years  
* `ratings` — user ratings (1–5 scale)

## Methodology

### Personalized Recommendation Approach

This assignment implements:

* **Item-to-Item Collaborative Filtering**

Key idea:

* Movies are compared based on **user rating patterns**
* Recommendations are generated by:
  * Finding similar movies
  * Using weighted averages of user-rated items

---

## Model Workflow

1. Load data from PostgreSQL
2. Clean and validate ratings
3. Create **train-test split (80/20)**
4. Construct **user-movie rating matrix**
5. Compute **cosine similarity between movies**
6. Predict ratings using item-based collaborative filtering
7. Evaluate predictions using RMSE and MAE
8. Generate **Top-3 recommendations per user**

---

## Evaluation

Model performance is evaluated using:

* **RMSE (Root Mean Squared Error)**
* **MAE (Mean Absolute Error)**

### Results:

* RMSE ≈ **0.308**
* MAE ≈ **0.308**

Due to the small and sparse dataset, results should be interpreted as a demonstration of recommender system logic rather than production-level performance.

## Key Insights

* Personalized recommendations differ from global baseline methods by incorporating **user-specific preferences**
* Small datasets can lead to:
  * Limited overlap between items
  * Similar predicted ratings across users
* Collaborative filtering still demonstrates meaningful recommendation logic even with limited data
